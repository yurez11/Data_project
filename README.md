##Проект 1: Анализ фильмов Pixar

**Цель:** Проанализировать успешность фильмов студии Pixar по кассовым сборам, оценкам критиков и зрителей, бюджетам и другим метрикам.

**Используемые технологии:** `pandas`, matplotlib.pyplot, seaborn

**Ключевые выводы и визуализации:**
1.  **Динамика мировых сборов:** Линейный график показывает рост и падение кассовых сборов по годам, помогая выявить наиболее успешные периоды для студии.
2.  **Сравнение рейтингов:** Точечная диаграмма выявляет корреляцию (или её отсутствие) между оценками профессиональных критиков (Rotten Tomatoes) и зрителей (IMDb).
3.  **География сборов:** Круговая диаграмма наглядно показывает, какая доля доходов приходится на внутренний (США/Канада) и международный рынки.
4.  **Эффективность бюджета:** Сдвоенная столбчатая диаграмма позволяет сравнить бюджет каждого фильма с его мировыми сборами, оценив окупаемость.
5.  **Динамика качества:** График средней оценки по годам (на основе Rotten Tomatoes и IMDb) показывает, как менялось восприятие фильмов с течением времени.
6.  **Влияние хронометража:** Исследована взаимосвязь между длительностью фильма и его финансовым успехом.
7.  **Распределение по рейтингам:** Анализ показывает, на какую целевую аудиторию (G или PG) в основном ориентируется Pixar.

##Проект 2: Анализ характеристик покемонов

**Цель:** Исследовать статистики покемонов (Pokémon) для выявления закономерностей между их типом, поколением и боевыми характеристиками.

**Ключевые выводы и визуализации:**
1.  **Сравнение типов по силе:** Столбчатая диаграмма ранжирует типы покемонов (Water, Fire, Dragon и др.) по средней атаке, выделяя самые "сильные" типы.
2.  **Эволюция силы по поколениям:** Линейный график показывает, как менялась средняя атака для каждого типа покемонов от поколения к поколению.
3.  **Распределение популяции:** Круговая диаграмма иллюстрирует соотношение покемонов разных типов в датасете.
4.  **Сравнительный анализ характеристик:** Сложная столбчатая диаграмма позволяет одновременно сравнить средние значения HP, Attack, Defense и других характеристик по поколениям.

##Проект 3: Анализ рейтингов аниме

**Цель:** Проанализировать базу данных аниме, чтобы определить самые популярные и высокооценённые тайтлы, а также исследовать зависимость рейтинга от популярности и типа контента.

**Ключевые выводы и визуализации:**
1.  **Топ-10 по рейтингу:** Горизонтальная столбчатая диаграмма наглядно представляет лучшие аниме согласно оценке сообщества.
2.  **Эффект популярности:** Точечная диаграмма исследует гипотезу о связи между количеством проголосовавших зрителей и средним рейтингом произведения.
3.  **Структура каталога:** Круговая диаграмма показывает распределение аниме по форматам (TV, Movie, OVA и др.).
4.  **Качество по форматам:** Столбчатая диаграмма сравнивает средний рейтинг для каждого типа аниме, выявляя, какой формат в среднем оценивается выше.
